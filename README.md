# E-commerce Empresax

Este é um **e-commerce** para a *empresa X*, feito em PHP e MySQL.

## Funcionalidades:

**Checkout, _Tela de produto_, Catáçogo, Home com banner**

###### Melhorias do projeto:

__Melhoria 1__, _melhoria 2_

### Linguagens do projeto:

* HTML
* CSS
* JavaScript
* PHP
* MySQL
